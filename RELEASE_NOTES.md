# RELEASE_NOTES.md

## v0.2 - 2026-06-26 保存点

本版本聚焦把「时空罗盘 / 湾区罗盘」从深圳核心演示扩展为更完整的大湾区文化探索产品，并补齐副本、印章、路线和素材治理闭环。

### 主要变化

- 锚点扩容到 64 个。
- 新增 30 个大湾区范围锚点，覆盖广州、佛山、肇庆、澳门、惠州、东莞、珠海、香港等区域。
- 所有锚点补齐剧情副本与印章。
- 副本题量提升：所有副本至少 2 题，重点锚点 3 题。
- 新增 `src/data/episode-expansions.js`，将副本扩展题库从主副本数据中拆出，便于后续维护。
- 印章册完成后可直接定位到刚获得印章，并显示「刚获得」/「当前印章」状态。
- 「剧情副本」入口重做为横向 CTA 卡片，去掉角色装饰，提升可读性与点击欲望。
- 「开始探索」重构为两个 tab：
  - 附近锚点 DIY 路线。
  - 热门推荐路线。
- 生成路线会展示在地图上，并提供大致距离与时间；实时导航跳转腾讯地图。
- 锚点详情模板去掉低价值标签和无意义模块，改为按内容类型展示「核心人物 / 主题关键词 / 文化地标」。
- 图片策略开始从生成横幅转向真实照片，已接入 16 个 Commons 实景图作为过渡。
